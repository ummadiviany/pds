Install Python

Python is a high-level, interpreted, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation. Python is dynamically-typed and garbage-collected.

Installing python easier than you can think of. But for stater this description might help

  1. Check if it already insalled. Type this command in the terminal: python --version or python3 --version If you see something like this : Python 3.9.0 or any other version then you are good to go. Not need to install. If you see something like this : python is not recognized ...., then you need to install.
  2. Download the latest Python 3.10.6 from Python download page and follow the instructions
  3. Install Python
    • Windows : Click one the downloaded file and proceed with installation steps. Mark the checkbox Add to Path and click Install Now. Check this tutorial here for more details.
    • macOS : Click on the downloaded file and proceed with installation steps. Keep all the options default.
    • Linux/Ubuntu : In the terminal type sudo apt-get install python3.

Install Visual Studio Code

VS Code is a free and open-source integrated development environment (IDE) for developing and debugging code. VSCode combines the simplicity of a source code editor with powerful developer tooling, like IntelliSense code completion and debugging. It support all the major programming languages.

Download the latest Visual Studio Code from Visual Studio Code download page and follow the instructions.

Windows/MacOS : Click on the downloaded file and proceed with standard installation.
Linux/Ubuntu : After downloading you have to run the following command in the terminal: sudo dpkg -i ~/path/to/code_1.XXX.deb. Or else you can also use sudo apt-get install code to install from the package manager.

Install Git

Git is a version control system. It is used to manage projects and to collaborate with other developers. In our case we are using it to manage our assignments and to submit them to the course repository. Just to let you know, your assignments are auto-graded, means that you will be graded on the basis of your passing test cases.

  1. Head to Git download page and download the git file.
  2. Click on the downloaded file and proceed with the installation steps. I'm sure you won't face any issues with this.

Creating your GitHub account

GitHub is a web-based hosting service for version control using Git. It is free and open source. You will be submitting your assignments to the course repository on GitHub.
This to keep in mind :

  • Other than being useful to our course, you will get a ton of benfits with your GitHub account. You can spin-up your personal website in minutes like mine ummadiviany.github.io. You can customize your own readme profile like this github.com/ummadiviany. Also check out GitHub Education benfits
  • Choose your GitHub username appropriately, such that it is readable and memorable for other fellows. Your typical username should be less than 15 characters. You can try firstname+lastname or just firstname.
  • Now you need to generate a Personal Access Token to use GitHub with your terminal/VS Code.
    On you GitHub account, follow these stepsSettings -> Developer Settings -> Personal access tokens -> Generate new token. Please save this token in a secure place on your machine. You will need it to configure your VS Code and Git. Check out this article if you need more details.

Configure Git with your credentials

Now you need to configure your Git with your GitHub credentials. Open your terminal/cmd.

  • Type git config --global user.name "your username" and replace "your username" with your username and hit enter. This command for me looks like this git config --global user.name ummadiviany
  • Type git config --global user.password "your password" and replace "your password"with peronal access token that you have generated. This command for me looks like this git config --global user.password your_personal_access_token

Configure VSCode for Python & Source Control

Now you need to configure VSCode for Python and Source Control(GitHub). Python is a community driven project and there are many extensions for VSCode. We will be installing a few of them.

  • Open VSCode -> Extensions -> Search for Python -> Install the recommended toolkit for Python by Microsoft . Check out this article if you need more help.
  • Open VSCode -> Extensions -> Search for GitHub -> Install GitHub by Knister Peter. Check out this article if you need more help.
